# My Projects

## Project 1: Snake Game

### Description

A console-based Snake game built in C#. The player controls a snake that moves around the screen eating food to grow longer. The game ends if the snake hits the wall or itself.

### Technologies Used

- C#
- Console Application
- Visual Studio
